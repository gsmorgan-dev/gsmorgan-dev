## 🚀 Featured Projects

### 🏋️ [LyftLogs](https://github.com/gsmorgan-dev/lyftlogs) — Workout Tracking Progressive Web App
> Node.js · Express · MongoDB · React · Docker

A full-stack PWA for tracking workouts, competing on leaderboards, and browsing exercise resources. Designed and built the MongoDB schema, REST API layer, and Docker infrastructure. Includes full offline support via service workers and the Cache API.

---

### 📊 [NYC Airbnb Market Analysis](https://github.com/gsmorgan-dev/nyc-airbnb-eda) — Exploratory Data Analysis
> Python · Pandas · NumPy · Matplotlib · Seaborn · Folium

An exploratory data analysis of 40,000+ NYC Airbnb listings sourced 
via the KaggleHub API. Performed systematic data cleaning, 
statistical analysis of pricing behavior across all five boroughs, 
and built interactive geospatial visualizations to surface market 
patterns and pricing drivers.
